Built-in server programming language

Macro

% (0x25) — basic prefix of macroes

& (0x26) — connect links _one*.htm

# (0x23) — insert scripts.js

1 (0x31) — insert _header.htm

2 (0x32) — insert dash panel _dash.htm

3 (0x33) — insert _menu.htm

4 (0x34) — insert _footer.htm

! (0x21) — insert IP address

? (0x3F) — insert library Flotr2

+ (0x2B) — insert library Processing

@ (0x40) — insert library Three

= (0x3D) — insert library jQuery

Info

Макрос состоит из двух символов — базового префикса ( % ) и уникального идентификатора (без пробела между ними). Идентификатор определяет операцию, которую нужно произвести в том месте файла, где он встречается. Это может быть вставка IP адреса системы, вставка кода библиотеки или любое другое заранее определённое действие.

Здесь представлены макросы, которые уже имеются в системе. Вы можете дополнить этот список любыми необходимыми вам действиями по вставке информации в передаваемые сервером файлы.
